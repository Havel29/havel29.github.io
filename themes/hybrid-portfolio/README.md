## Installation

1. Clone or download this theme to your Hugo site's `themes` directory:

```bash
git clone https://github.com/yourusername/hybrid-portfolio.git themes/hybrid-portfolio
```

2. Update your site's `hugo.toml` configuration:

```toml
theme = "hybrid-portfolio"
```
